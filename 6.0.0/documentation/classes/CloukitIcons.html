projects/cloukit/icon/src/lib/icon.paths.ts

Description

An icon consists of exactly one SVG path inside a 512x512 pixel viewbox. If you want to inject your own icons keep that in mind. Draw your icons in any way and use "combine-path" or "convert to outlines" tools to create a single path.

Your icons svg code should look like this:

<svg viewBox="0 0 512 512">
     <path d="M23233....." ></path>
</svg>

We just need the value of the path "d" attribute.
