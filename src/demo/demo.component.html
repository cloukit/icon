icon transformation

Use the transform attribute to be applied to any icon to change its size, rotation or position. Note that the viewport is always 512x512 pixel.
<cloukit-icon
  combine="stack"
  foregroundIcon="check" foregroundIconColor="#3FA051"
  foregroundIconTransform="rotate(45, 256, 256), scale(0.7), translate(100, -130)"
  backgroundIcon="triangle" backgroundIconColor="#0f0eff"
></cloukit-icon>
